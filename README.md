# Primeiros Passos em Ciência da Computação (2024)

Este repositório contém meus primeiros projetos desenvolvidos em **Linguagem C**. 
O objetivo foi aplicar conceitos básicos de lógica, como:
* Estruturas de decisão (if/else)
* Laços de repetição (while/for)
* Manipulação de variáveis e tipos de dados

## 📂 Projetos incluídos:
* **Concessionária:** Gerenciamento simples de estoque ou vendas.
* **Petshop:** Sistema de controle de serviços ou cadastros.
* **Cálculo de Média:** Exercício fundamental de aritmética e entrada de dados.
